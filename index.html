🌿 Sobre o Projeto

Uma solução de software livre desenvolvida por estudantes do IFTM para preservação do Parque do Goiabal em Ituiutaba-MG

🎯 Objetivo

Desenvolver uma plataforma web de software livre para registro da biodiversidade e denúncia de problemas ambientais no Parque do Goiabal, aumentando o engajamento da comunidade e a capacidade de monitoramento ambiental da área.

📜 Licença GNU GPL v3

⚙️ Tecnologias

  • HTML5, CSS3 e JavaScript
  • Backend em Python
  • Banco de dados relacional
  • Controle de versão via GitHub
  • Hospedagem com software livre

🎯 Metas de impacto

  • 100 usuários cadastrados no 1º mês
  • 50 registros de biodiversidade
  • Redução no tempo de acionamento de bombeiros
  • Integração com comunidade local e UFU

🔮 Futuro

  • Aplicativo mobile (fase 2)
  • Integração com sistemas municipais
  • Expansão para outras áreas verdes
  • Parceria com órgãos ambientais

🤝 Colaboração com a comunidade

O projeto foi desenvolvido em parceria com moradores de Ituiutaba e visitantes do parque, com entrevistas e testes realizados junto à comunidade. Também contamos com apoio de estudantes da Universidade Federal de Uberlândia (UFU), que desenvolvem projetos paralelos de preservação da área. A comunidade participa ativamente através de registros e denúncias na plataforma.
